# Chatbot Project

A React-based chatbot application with an interactive UI.

## Features
- Real-time message sending and receiving
- Auto-scrolling chat interface
- Loading animations
- User and robot message differentiation
- Clean, responsive design

## Files
- `index.html` - Main HTML file with React JSX code
- `robot.png` - Robot avatar image
- `user.png` - User avatar image
- `loading-spinner.gif` - Loading animation

## Technologies Used
- React.js
- Babel (JSX transpiler)
- CSS3

## How to Run
1. Open `index.html` in a web browser
2. Start typing messages and click "Send" or press Enter
3. The chatbot will respond in real-time
